# Source Discipline

## Source Note Schema

Each source entry should record:

- title;
- author/organisation;
- publication or update date;
- date accessed;
- URL;
- source type;
- key claims;
- direct quotation under 25 words, if useful;
- relevance to Government / Local Authority AI Hub decision-making;
- reliability assessment;
- sections where this source may be cited.

## Use Rules

- A fact can be drafted only when a source note supports it.
- A reported opinion must identify who is speaking and why the source matters.
- Analysis may reason from evidence but must not smuggle in new claims.
- Vendor research should be paired with primary, academic, or independent evidence before final recommendations rely on it.

## Imported Report Handling

The imported Deep Research report is useful for orientation and gap-finding. It is not a final citation source. Convert any useful claim into a structured source note with a durable URL and citation key first.
