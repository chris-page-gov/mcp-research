# Imported Deep Research Artefacts

This directory contains raw imported artefacts from the first Deep Research attempt.

## Files

- `MCP as an integration backbone for enterprise and government AI.md`: imported report text for orientation and gap-finding.
- `Citations from MCP as an integration backbone for enterprise and government AI`: imported citation export for source-lead recovery.
- `Activity from MCP as an integration backbone for enterprise and government AI`: raw research activity and process notes.

## Use Rules

- Do not cite files in this directory directly in final paper prose.
- Convert useful claims or leads into structured entries under `sources/`.
- Treat the activity file as audit trail and source-lead context, not evidence.
- Keep sensitive or private material out of this directory because the repository is public.
